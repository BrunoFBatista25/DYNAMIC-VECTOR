# Vector Dinâmico em C

Uma implementação genérica de vetor dinâmico (similar ao `std::vector` do C++) feita em C puro com boa arquitetura.

## Características

- Encapsulamento completo (pseudo-POO)
- Crescimento automático exponencial
- Suporte a qualquer tipo de dado (`int`, `double`, `struct`, etc.)
- Gerenciamento correto de memória
- Separação clara entre interface e implementação

## Como compilar e executar

```bash
make
./vector_test
